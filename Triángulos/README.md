Los azucareros del centro necesitan escribir un programa triangles para calcular el numero de triangulos diferentes con perimetro P tal que las longitudes de los lados sean enteros cuando P es un entero.



## Entrada



Leer en una linea simple el valor de P (2 < P  < 100000).



## Salida



La unica linea de la salida debe contener el numero de triangulos calculados.



## Ejemplo de Entrada



```
5
```


## Ejemplo de  Salida



```
1
```
