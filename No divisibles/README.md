?Cuantos numeros hasta N no se dividen por 2, 3 o 5?



## Entrada



La entrada consiste en un numero entero 1 \leq N \leq 10\,000\,000.



## Salida



Imprimir en una linea la cantidad de numeros que cumplen lo antes descrito.



## Ejemplo de entrada



```
10
```


## Ejemplo de salida



```
2
```
