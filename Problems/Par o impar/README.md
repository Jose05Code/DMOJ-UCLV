Aqui va otro problema trivial.



Dado un un numero entero N con -100\,000 \leq N \leq 100\,000, debera decir si el mismo es par (even) o impar (odd).



## Entrada



El numero N.



## Salida



Debera imprimir odd si es impar o even si es par.



## Ejemplo de entrada



```
387
```


## Ejemplo de salida



```
odd
```
